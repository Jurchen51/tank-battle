# 🎮 坦克大战 - Tank Battle

一款基于 HTML5 Canvas 的本地双人对战坦克游戏。

## 🕹️ 游戏玩法

- **双人对战**：两名玩家在同一键盘上对战
- **5局3胜制**：先赢得3局者获得最终胜利
- **多种武器**：炮弹 + 地雷
- **可破坏地形**：砖墙可被炮弹摧毁，钢墙不可摧毁
- **道具系统**：生命恢复、加速、连射、护盾
- **小地图**：右下角实时显示战场态势

## 🎯 操作说明

| 操作 | 玩家1 (🔵) | 玩家2 (🔴) |
|------|-----------|-----------|
| 移动 | W A S D | ↑ ← ↓ → |
| 射击 | F | / |
| 埋雷 | G | . |
